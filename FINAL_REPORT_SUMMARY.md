# Final Report Summary

## System Overview

The Type2 system is a deterministic-first physics QA solver. It parses the problem, retrieves formulas, constructs a stepwise trace, verifies intermediate steps, and produces a final answer with optional answer-level validation metadata. The hybrid path keeps the deterministic solver as the first pass and calls the local vLLM-backed DSPy fallback only when deterministic solving fails.

The current local model configuration uses an open-source <=8B model served by vLLM through an OpenAI-compatible endpoint. No closed-source API dependency is required for final inference.
